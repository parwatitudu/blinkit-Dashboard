# blinkit-Dashboard
## Project Overview
This project analyzes blinkit sales data to understand customer behavior and sales trends.

## Tools Used
-Power BI
-Excel

## Key Insights
-Monthly order trends show variation in customer demand over time
-Revenue trends highlights peak sales periods
-Top 5 areas contribute the highest share of total sales
-Payment mode analysis shows preferred payment methods among customers

## Conclusion
The dashboard helps identify sales trends,high-performing locations,and customer preferences,enabling
better business decision-making.
